Mysql数据库的异常

发布于 2020-03-27 | 分类于 数据库 | 1分钟 | 143字数

好久没有用数据库了，今天因为学习上需要使用，登录发现服务未打开，去服务中找了一下，我的数据库服务呢？？？丢了？？？
在百度后发现需要把服务安装一下。
1、cmd进入dos窗口找到自己的mysql装入的位置，进入到bin文件夹下面。我的路径：C:\Program Files (x86)\MySQL\MySQL Server 5.5\bin。
2、输入mysqld -instatll命令，如果显示Service successfully installed就安装成功，（注：需要以管理员身份运行）。
3、net start mysql 启动服务。
成功解决。
