[蓝桥杯刷题]：IP判断

  1. 2020-02-21
  2. 388字
  3. 2 min read

题目

题目描述

在基于Internet的程序中，我们常常需要判断一个IP字符串的合法性。
合法的IP是这样的形式：
A.B.C.D
其中A、B、C、D均为位于[0, 255]中的整数。为了简单起见，我们规定这四个整数中不允许有前导零存在，如001这种情况。
现在，请你来完成这个判断程序吧_

输入

输入由多行组成，每行是一个字符串，输入由“End of file”结束。
字符串长度最大为30，且不含空格和不可见字符

输出

对于每一个输入，单独输出一行
如果该字符串是合法的IP，输出Y，否则，输出N

样例输入

1.2.3.4
a.b.c.d
267.43.64.12
12.34.56.bb
210.43.64.129
-123.4.5.6

样例输出

Y
N
N
N
Y
N

解题

思路：

这道题难点在如何以点把一串ip地址分隔成四段，如果用char类型判断时需要在转换为int，很麻烦，所以引入了sscanf()。
关于sscanf()可以看这篇文章。

参考代码

#include<cstdio>
#include<cstring>
#include<stdlib.h>
bool judge(int a){
	if(a >= 0 && a <= 255){
		return true;
	}
	return false;
} 
int main(){
	
	char st[16];
	int a, b, c, d;
	
	while(scanf("%s", &st) != EOF){
		if(sscanf(st,"%d.%d.%d.%d",&a, &b, &c, &d) == 4){
			if(judge(a) && judge(b) && judge(c) && judge(d)){
				printf("Y\n");
			}else{
				printf("N\n");
			}
			
		}else{
			printf("N\n");
		}
	}
	
	
	
	return 0;
}
